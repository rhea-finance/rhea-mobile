🔗 Rhea Bridge Demo

Web ↔ Native 通信调用示例

🌐 浏览器环境（模拟模式）
📄 打开二级页面（Navigate）
自动读取页面标题
不传 title 参数，RN 端会使用当前 document.title 作为导航栏标题
RheaBridge.navigate('https://x.rhea.finance/supply/usdc')
指定自定义标题
传入 title 参数，RN 端使用该标题
RheaBridge.navigate( 'https://x.rhea.finance/supply/usdc', 'Supply USDC' )
📦 打开底部 Modal（OpenModal）
加载 URL 内容
Modal 内嵌 WebView 加载指定 URL
RheaBridge.openModal({ title: 'Supply USDC', url: 'https://x.rhea.finance/modal/supply' })
传递数据对象
将 token 信息、APY 等业务数据传递给 Modal 内的页面
RheaBridge.openModal({ title: 'Supply Details', url: 'modal-content.html', data: { token: 'USDC', apy: '5.2%', tvl: '$12.5M', chain: 'Ethereum' } })
仅传数据（无 URL）
只传 data，不传 url。RN 端可自行决定如何渲染内容
RheaBridge.openModal({ title: 'Token Info', data: { token: 'ETH', price: '$3,200', change: '+2.5%' } })
✖️ 关闭 Modal（CloseModal）
程序化关闭
在 Modal 内的页面中，调用此方法可以主动关闭 Modal
RheaBridge.closeModal()
🚀 完整对接代码参考
在你的 Web 页面中使用
<script> // Bridge 注入极快，通常页面加载后即可直接调用 function onSupplyClick(token) { RheaBridge.openModal({ title: 'Supply ' + token.symbol, url: '/modal/supply?token=' + token.address, data: token }); } </script>
